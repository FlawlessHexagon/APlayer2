# APlayer2

A cross-platform local music player rebuilt from the ground up.

**Framework:** Flutter (Dart)
**Core Architecture:** Native C++ DSP Audio Engine via `dart:ffi`
**Target Platforms:** macOS, Android (Launch) | Windows, Linux, iOS, Web (Future)

## Documentation
All project planning, architecture decisions, and agent protocols are located in the `_doc/` directory.

- `_doc/doc_general.md` - Core project plan and technical direction.
- `_doc/dev_phase/` - Detailed phase-by-phase implementation plans.
